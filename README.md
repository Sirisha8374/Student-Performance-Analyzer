Description
-----------
A Python program that analyzes student internal marks stored in a list.
Each mark is processed using a for loop and classified into performance categories.
The program also counts total valid students and total failed students.

Classification Rules
--------------------
90–100 → Excellent
75–89 → Very Good
60–74 → Good
40–59 → Average
0–39 → Fail
<0 or >100 → Invalid

Approach
--------
Use list to store marks Process each mark using a for loop Apply conditional statements for classification Maintain counters for valid and failed students

Output
------
Displays adjusted mark with category Prints: Total Valid Students Total Failed Students

Language
--------
Python
